# Configuration Examples

This folder contains example configurations for different project types.

## Available Examples

### 1. Minimal TypeScript Project
A minimal configuration for a TypeScript/Node.js project with:
- Basic LSP configuration
- Type checking hooks
- ESLint integration

### 2. Full-Stack Web App
Configuration for a Next.js/React web application with:
- Frontend and backend LSP
- Database migration hooks
- Deployment workflow integration

### 3. Python Data Science
Configuration for Python/Jupyter projects with:
- Python LSP and linting
- Jupyter notebook support
- Data science tool integration

## Using Examples

```bash
# Copy an example configuration to your project
cp -r examples/minimal-typescript/.claude/ ~/your-project/

# Customize for your needs
edit ~/your-project/.claude/settings.json
```
